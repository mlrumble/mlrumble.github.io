How I created this website using Hugo and used a custom domain with github pages

  • Nov 27, 2017
  • 2 min read

The first Step:

Go to the quickstart tutorial and follow through with the steps. The main steps are:

brew install hugo
# replace with your site name
hugo new site mlrumble
cd mlrumble;\
git init;\
git submodule add https://github.com/avianto/hugo-kiera.git themes/kiera
# create a new post
hugo new posts/my-first-post.md
# run the server with draft mode enabled
hugo server -D

Once done you can see your website up and running at localhost:1313

Adding Social Buttons, Disqus IDs, or Google Analytics:

Next step is to change the config.toml file to this if you want to use the kiera theme. This step is mainly a little theme specific:


baseURL = "http://mlrumble.github.io"
languageCode = "en-us"
title = "MLRumble"
theme = "kiera"

pygmentsCodeFences = true

disqusShortname = "mlrumble" #Disqus shortname
googleAnalytics = "" #Google Analytics ID

[author]
    name = "MLRumble"       #Author name
    github = "mlrumble"     #Github username
    gitlab = ""     #Gitlab username
    linkedin = ""   #LinkedIn username
    facebook = ""   #Facebook username
    twitter = "mlrumble"    #Twitter username
    instagram = ""  #Instagram username

[params]
    tagline = "the tagline for this website"

Adding Pages:

if you want to add a page about for your website add about.md in the content directory


+++

title = "About Hugo"
date = "2014-04-09"
menu = "main"
weight = "20"
meta = "false"
+++
